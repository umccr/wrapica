Welcome to wrapica's documentation!
===================================

**wrapica** is a Python library that uses the `libica API <https://umccr.github.io/libica/openapi/>`_
to wrap useful combinations of ICAv2 API commands together

Check out the `modules <modules.html>`_ section for further information on using the python functions available.

If you are using BYOB configurations within your ICAv2 Projects (or external S3 storage credentials), please consult the `byob configuration <byob_configuration.html>`_ setup for more information

.. note::

   This project is under active development.
