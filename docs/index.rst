Welcome to wrapica's documentation!
===================================

**wrapica** is a Python library that uses the `libica API <https://umccr-illumina.github.io/libica/openapi/>`_
to wrap useful combinations of ICAv2 API commands together

Check out the `modules <modules.html>`_ section for further information on using the python functions available.

.. note::

   This project is under active development.
